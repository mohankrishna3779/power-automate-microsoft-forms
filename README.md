# Power Automate – Microsoft Forms Integration
## Overview
This repository contains an exported Power Automate flow triggered by Microsoft Forms.
## Flow Name
Employee Consent Flow
## Trigger
Microsoft Forms – When a new response is submitted
## Purpose
- Capture employee consent data
- Process form responses using Power Automate
## How to Use
1. Download the ZIP file from this repository
2. Go to Power Automate
3. Click Import
4. Upload the ZIP file
5. Configure required connections
6. Save and Test
## Tools Used
- Microsoft Forms
- Power Automate
- GitHub
